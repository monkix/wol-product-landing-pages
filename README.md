# Wizards of Learning Product Landing Pages

Static landing pages for Wizards of Learning products.

## Pages

- `index.html` - Wizards of Learning service landing page
- `fish-farmer.html` - Fish Farmer vs Alien Species product page
- `tarot-of-learning.html` - Tarot of Learning product page

## Local Preview

```bash
python -m http.server 8123
```

Then open:

- `http://127.0.0.1:8123/fish-farmer.html`
- `http://127.0.0.1:8123/tarot-of-learning.html`

## Deployment

This site is static HTML/CSS and can be published with GitHub Pages.
